# Images Folder

Place your images here.

## Recommended Images

1. **Profile Picture**: `profile.jpg` or `profile.png`
   - Recommended size: 400x400px or square aspect ratio
   - For the About section

2. **Project Images**: `projects/project1.jpg`, `projects/project2.jpg`, etc.
   - Recommended size: 800x600px or 16:9 aspect ratio
   - For the Projects section

3. **Hero Background**: Optional if not using video
   - Recommended size: 1920x1080px or 16:9 aspect ratio

## Image Optimization

- Use WebP format for better compression
- Compress images to reduce file size
- Use appropriate dimensions for each use case
